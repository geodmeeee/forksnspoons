# Ace Attorney Webport

A **web port of Ace Attorney** playable in your browser.

This repository contains the source for a browser‑based port of the classic Ace Attorney court games. All copyrighted game content (characters, music, dialogue, art) belongs to **Capcom and Nintendo** — this project is a fan‑made port and not an official release.

## What This Is
- A **client‑side web port** of Ace Attorney  
- Works in modern browsers  
- Uses HTML, CSS, and JavaScript for UI and interactivity  
- Designed for fans to play Ace Attorney without installing anything
